# Метаэвристики

Метаэвристикой называют процедуру для создания эвристик поиска оптимальных решений в задачах оптимизации. Чаще всего метаэвристики налагают минимальные условия на целевую функцию и ограничения в задаче оптимизации, что позволяет их использовать для широкого класса задач.

## Локальный поиск через hill climbing

Зададимся начальным приближением $\boldsymbol{x}^{(0)}$.
Hill climbing предполагает наличие процедуры `tweak()`, которая генерирует $N$ локально измененных копий $\boldsymbol{x}^{(0)}$.
Под "локальностью" подразумевается небольшой характер вносимых изменений.
Например, если $\boldsymbol{x}^{(0)} \in \mathbb{R}^K$, то `tweak()` может изменить значение одной из компонент вектора.
Далее из $N$ локально измененных копий и $\boldsymbol{x}^{(0)}$ выбирается наилучшее, после чего процедура повторяется.

\begin{align}
\boldsymbol{x}_1, ..., \boldsymbol{x}_N &= \text{tweak}(\boldsymbol{x}^{(k)}), \\
\boldsymbol{x}^{(k+1)} &= \text{argmax}_{\boldsymbol{x} \in \{\boldsymbol{x}_1, ..., \boldsymbol{x}_N, \boldsymbol{x}^{(k)}\}} f(\boldsymbol{x}).
\end{align}

## Раскраска вершин графа с помощью hill climbing

Задача раскраски вершин графа состоит в назначении каждой вершине одного из C цветов так, чтобы не нашлось ни одной пары соседей с одинаковыми цветами.
Простейшая реализация процедуры tweak в hill climbing состоит в случайном изменении цвета одной из случайно выбранных вершин.
